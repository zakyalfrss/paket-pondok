# FOTO UNTUK NOTIFIKASI WHATSAPP

## CARA MENAMBAH FOTO:
1. Simpan foto di folder ini (assets/images/)
2. Format nama file:
   - makanan-cepat-basi.jpg (untuk barang cepat basi)
   - makanan-biasa.jpg (untuk barang normal)
   - default.jpg (jika file spesifik tidak ada)

3. WhatsApp akan otomatis kirim foto sesuai jenis barang

## CONTOH:
- makanan-cepat-basi.jpg → untuk notifikasi barang cepat basi
- minuman.jpg → untuk notifikasi minuman
- default.jpg → untuk semua jenis barang lainnya
